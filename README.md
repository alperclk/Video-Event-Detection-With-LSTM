# Video-Event-Detection-With-LSTM
Video Classification with lstm architecture

Requirements library
OpenCV
Numpy
Keras,Tensorflow

LSTM architecture was used in our study.
There are 10 goals and 10 other events in the "dataset" folder of the training videos.
The "records" file is where the predictions of the model are to be recorded.
Since the training of the model takes a long time, the model named "model" can be used.
To do this, simply run the "predict.py" or "predict.ipynb" python file.

------------------------------------------------------------------------

Gerekli kütüphaneler
OpenCV
Numpy
Keras,Tensorflow

Çalışmamızda tekrarlayan sinir ağı modellerinden LSTM mimarisi kullanılmıştır.
Eğitim videoları "dataset" klasöründe 10 gol, 10 diğer olaylar olarak bulunmaktadır.
"records" dosyası ise modelin tahminlerinin kaydedileceği yerdir.
Modelin eğitimi uzun zaman sürdüğünden, kaydedilmiş olan "model" adındaki model kullanılabilir.
Bunun için yalnızca "predict.py" veya "predict.ipynb" python dosyasının çalıştırılması yeterlidir.
